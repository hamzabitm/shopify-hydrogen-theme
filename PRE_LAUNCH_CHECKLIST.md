## ‚ö†Ô∏è Common Issues & Solutions

**Issue**: Port 3000 already in use
**Solution**: Dev server auto-selects next available port (3001, 3002, etc.)

**Issue**: CSS not updating
**Solution**: Clear browser cache and hard refresh (Ctrl+Shift+R)

**Issue**: Images not loading
**Solution**: Check file paths and ensure files exist in public/ folder

**Issue**: GraphQL errors
**Solution**: Verify store is linked with `h2 link` command

**Issue**: Checkout not working
**Solution**: Ensure payment gateway is configured in Shopify
